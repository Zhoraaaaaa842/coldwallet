# ZhoraWallet — Холодный кошелёк Ethereum

Программный cold wallet с air-gapped подписью транзакций через USB.

---

## Архитектура

```
┌─────────────────┐     USB      ┌──────────────────┐
│  Онлайн-ПК       │◄──────────►│  Офлайн-ПК          │
│  (ZhoraWallet)   │   pending/  │  (SignOffline.exe)  │
│                  │   signed/   │                    │
│  • Баланс        │              │  • Приватный ключ │
│  • Создание TX  │              │  • Подпись TX     │
│  • Broadcast      │              │  • Мнемоника      │
└─────────────────┘              └──────────────────┘
```

Приватный ключ **никогда не покидает** офлайн-среду.

## EXE-файлы

После сборки `python build_all.py` в папке `dist/ColdVault_Setup/` появляются:

| Файл | Назначение |
|---|---|
| `ZhoraWallet.exe` | Десктоп-приложение кошелька (онлайн-ПК) |
| `ZhoraUSB.exe` | GUI-установщик холодного кошелька на флешку |
| `USB_Files/SignOffline.exe` | Утилита офлайн-подписи (офлайн-ПК) |
| `INSTALL.bat` | Меню-мастер установки |

```
dist/ColdVault_Setup/
├── ZhoraWallet.exe      ← десктоп-кошелёк
├── ZhoraUSB.exe         ← установщик на флешку
├── INSTALL.bat          ← меню-запускальщик
├── USB_Files/
│   └── SignOffline.exe  ← копируется на флешку при установке

```

## Быстрый старт

### 1. Зависимости

```bash
pip install -r requirements.txt
```

### 2. Установка кошелька на USB

**Вариант A — через `ZhoraUSB.exe` (рекомендуется)**

1. Запустите `ZhoraUSB.exe` от имени администратора (UAC-запрос появится автоматически)
2. Выберите флешку из выпадающего списка
3. Выберите: форматировать или нет
4. Нажмите **«⚡ Установить на флешку»** — подтвердите диалог
5. ZhoraUSB автоматически скопирует `SignOffline.exe` на флешку

**Вариант B — через командную строку**

```bash
python installer/install_to_usb.py
```

> ❗ Запускайте на **офлайн-ПК** для максимальной безопасности.

#### Установка на вторую флешку

Каждая флешка — **отдельный независимый кошелёк** (разные ключи и адреса).  
Для резервной копии (тот же адрес) — при установке выберите «Импортировать» и введите мнемонику от первой.

### 3. Запуск десктоп-приложения (онлайн-ПК)

**Через EXE:**
```
ZhoraWallet.exe
```

**Через Python:**
```bash
python run_desktop.py
```

При первом запуске:
1. Подключите USB с кошельком
2. Нажмите **«Обнаружить USB»** — приложение найдёт `wallet.vault`
3. Введите пароль — кошелёк разблокируется
4. Адрес и баланс появятся на главном экране

### 4. Сборка EXE

```bash
python build_all.py
```

Флаги (необязательно):

| Флаг | Что собирает |
|---|---|
| `--desktop-only` | Только `ZhoraWallet.exe` |
| `--signer-only` | Только `SignOffline.exe` |
| `--installer-only` | Только `ZhoraUSB.exe` |
| `--no-clean` | Не удалять предыдущую сборку |

## Рабочий процесс

### Отправка ETH (полный цикл)

```
[Онлайн-ПК]  →  [USB]  →  [Офлайн-ПК]  →  [USB]  →  [Онлайн-ПК]
ZhoraWallet    pending/  SignOffline.exe   signed/   ZhoraWallet
Создать TX              Подписать               Broadcast
```

**Подробно:**

1. **Онлайн-ПК** (`ZhoraWallet.exe`) → вкладка **«Отправить»**
   - Введите адрес, сумму, gas → **«Создать TX → USB»** → файл в `pending/`
2. Извлечь USB, вставьте в офлайн-ПК
3. **Офлайн-ПК** (`SignOffline.exe`) → сканировать `pending/` → подписать TX → файл в `signed/`
4. Вставьте USB обратно в онлайн-ПК
5. **Онлайн-ПК** (`ZhoraWallet.exe`) → сканировать `signed/` → **«⯈ Отправить в сеть»**

### Получение ETH (QR-код)

1. Вкладка **«Получить (QR)»** → **«Сгенерировать QR»**
2. Покажите QR отправителю или сохраните в PNG

## Структура USB

```
USB/
├── SignOffline.exe        # утилита офлайн-подписи
├── launch_signer.bat      # быстрый запуск (Windows)
├── autorun.inf            # метка диска
├── README.txt
└── ColdVault/
    ├── wallet.vault           # Зашифрованный кошелёк (AES-256-GCM)
    ├── config.json            # Конфигурация (chain_id, сеть)
    ├── pending/               # Неподписанные TX
    ├── signed/                # Подписанные TX
    └── tools/                 # запасные Python-скрипты
```

## Сценарии с несколькими флешками

| Сценарий | Действие |
|---|---|
| Новый независимый кошелёк | `ZhoraUSB.exe` или `install_to_usb.py` → новый пароль |
| Резервная копия (тот же адрес) | `install_to_usb.py` → «Импортировать» → мнемоника от первой |
| Разные адреса для разных целей | `ZhoraUSB.exe` на каждую флешку отдельно |

## Безопасность

| Компонент | Технология |
|-----------|-----------|
| Шифрование ключа | AES-256-GCM |
| Деривация пароля | PBKDF2-SHA256 (600 000 итераций) |
| Мнемоника | BIP-39 (24 слова, 256 бит) |
| HD-деривация | BIP-44 (m/44'/60'/0'/0/0) |
| Транзакции | EIP-1559 + Legacy, ECDSA secp256k1 |
| QR-формат | EIP-681 |

Полный аудит: [SECURITY_AUDIT.md](SECURITY_AUDIT.md)

## Предупреждение

Это **программный** холодный кошелёк. Он не заменяет аппаратные кошельки (Ledger, Trezor). Используйте для обучения и умеренных сумм. Для крупных сумм используйте аппаратный кошелёк.
